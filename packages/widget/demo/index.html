Qelp widget demo

This page embeds the Qelp feedback widget. Click the floating "Send feedback" button to capture a screenshot, record a voice note, and submit feedback to your local API.

How to embed

<script src="/qelp-widget.js"></script>
<script>
  QelpWidget.init({
    apiBaseUrl: "http://localhost:4000",
    workspaceId: "ws_studio",
    projectId: "proj_meteor",
    projectKey: "widget_live_meteor",
    themeColor: "#0f766e"
  });
</script>

For local testing, copy dist/qelp-widget.js next to this HTML file (or serve them from the same static server) so the script tag below can find it.

Why this works

The widget uses html2canvas for screenshots, the MediaRecorder API for voice notes, and uploads each blob to POST /uploads on the API. The feedback POST then references the attachment IDs.
